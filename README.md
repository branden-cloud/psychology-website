# Psychology Practice Website

A responsive website designed for a psychology and mental health practice.

The goal of this project was to create a clean, professional layout that clearly presents services, resources, and contact information while maintaining an approachable and calming design.

## Features
- Responsive layout for mobile and desktop
- Modern UI design
- Clean content structure
- Service and contact sections

## Technologies
- HTML
- CSS
- JavaScript

## Purpose
This project demonstrates my ability to design and build professional service-based websites while continuing to develop my front-end and UI skills.
